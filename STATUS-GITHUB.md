## 🆘 Si Algo Sale Mal

### Problema: "Permission denied" en push
**Solución:** Configura el token de GitHub en tu credencial manager de Windows

### Problema: "remote: Repository not found"
**Solución:** Verifica que el repositorio existe en GitHub y la URL es correcta

### Problema: "fatal: Not a git repository"
**Solución:** Asegúrate de estar en la carpeta correcta (verifica con `pwd`)

### Problema: Archivos que no quería se subieron
**Solución:** 
```powershell
git rm --cached archivo-no-deseado.js
git commit -m "Remove archivo-no-deseado.js"
git push
```
